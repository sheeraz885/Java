Chapter 1

Alert()

Basic Information

    Basic Usage:
  • The alert function displays a message in a pop-up box.
  • Example: alert("Thanks for your input!");
    • Syntax:
  • Use lowercase alert (capitalizing it will cause an error).
  • Always include parentheses () after alert.
  • The message inside is a string, which can be enclosed in either double (") or single (') quotes, but we'll use double quotes for consistency.
    • Spacing:
  • No spaces should be placed before or after the parentheses.
  • Incorrect: alert ( "message" );
    • Ending Statements:
  • Each statement should end with a semicolon ; to maintain clarity, even if itâ€™s not always required.
    • Window Object:
  • You can also write window.alert, but alert is preferred for brevity.
