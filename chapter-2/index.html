Chapter 2

VARIABLES FOR STRINGS

Variables

  • Definition: A variable is a storage location in your program that holds a value. It has a name (identifier) and can be changed during the program's execution.
  • Example: In JS, let age = 25 creates a variable named age that holds the value 25.

Strings

  • Definition: A string is a sequence of characters, used to represent text. Strings are usually enclosed in quotes (single or double).
  • Example: name = "Sheeraz" creates a string variable named name containing the text Sheeraz.
